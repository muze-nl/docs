# Muze Javascript Libraries Reference

The Muze Javascript libraries are a collection of single purpose software components for workign with SOLID and linked data.

Here you will find the reference documentation for the following components:

- [**JAQT**, a Javascript library containing a query engine for arrays and objects](jaqt.md)
- [**JSFS-Solid**, a Javascript library offering client to access Solid pods through a filesystem-like interface.](JSFS-solid.md)
- [**MetroJS**, a javascript library containing an HTTPS client with support for middleware.](metroJS.md)
    - [**Metro OpenID Connect**, Middleware for MetroJS allowing the Metro client to handle authorisation and authentication using OpenID Connect.](metro-oidc.md)
    - [**Metro OAuth2**, Middleware for MetroJS allowing the Metro client to handle OAuth2 connections.](metro-oauth2.md)
- [**Object Linked Data Mapper**, also referred to as OLDM, is a library which maps Linked Data into Javascript objects.](OLDM.md)
